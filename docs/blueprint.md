# **App Name**: Techno Gym

## Core Features:

- Home Dashboard: Display member's QR code status, progress bar showing 'Days Remaining', and a 'Quick Scan' button.
- Subscription Management: Show membership tiers with pricing, benefits, and 'Renew Now' call-to-action.
- Workout Library: Categorized cards for Strength, HIIT, Cardio, Recovery with thumbnail placeholders for videos.
- Trainer Profiles: Display trainer avatars, specialties, and 'Book Session' button.
- Gym Shop Stock Check: List products (Proteins, Pre-workouts, Water) with stock status and price tags.
- Feedback Collection: Form for suggestions and 'Rate your last workout' star system.

## Style Guidelines:

- Background color: Dark gray (#121212) for a sleek dark mode theme.
- Primary color: Neon Blue (#00FFFF) to give a futuristic feeling.
- Accent color: Cyan (#008080) to complement the neon blue and add visual interest.
- Body and headline font: 'Inter', a grotesque-style sans-serif, offering a modern and neutral aesthetic suitable for both headlines and body text.
- Use modern, minimalist icons in neon blue and cyan to match the theme.
- Use a grid-based layout with clearly defined sections and whitespace for a clean, modern feel.
- Subtle animations and transitions when navigating between screens and updating content.
